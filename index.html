Keyboard Shortcuts & Help

Playback

  • Space - Play/Pause
  • ← / → - Step backward/forward
  • Shift + ← / → - Step 1 second
  • A / D - Previous/Next annotation

Tools

  • S - Select tool
  • M - Pin tool
  • B - Draw tool (or hold D)
  • Shift + Click - Quick pin

Timeline

  • I / O - Set In/Out points
  • X - Clear range
  • Alt + Drag - Pan timeline
  • Shift + Drag marker - Move annotation

Files

  • Ctrl + S - Save project
